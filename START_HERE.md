## 🚀 Швидкий старт (2 хвилини)

### Windows:
```cmd
1. Розпакуйте архів
2. Відкрийте командний рядок у папці проекту
3. Запустіть: npm install
4. Потім: npm run dev
5. Відкрийте: http://localhost:3000
```

### Mac/Linux:
```bash
# 1. Розпакуйте архів
tar -xzf kino-ua-project-v1.tar.gz
cd kino-ua-project

# 2. Запустіть автоматичний скрипт
./quick-start.sh

# АБО вручну:
npm install
npm run dev
```
